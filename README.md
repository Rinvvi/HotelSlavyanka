# HotelSlavyanka

Информационная система управления гостиницей «Славянка».

## Описание проекта

Приложение предназначено для автоматизации работы гостиницы и позволяет:

* выполнять авторизацию пользователей;
* разграничивать права доступа по ролям (администратор, сотрудник ресепшена);
* управлять номерным фондом;
* заселять и выселять гостей;
* вести базу данных гостей;
* хранить информацию о бронированиях;
* хранить информацию о сотрудниках;
* вести учет пользователей системы.

## Используемые технологии

* C#
* WPF
* ASP.NET Core Web API
* Entity Framework Core
* PostgreSQL
* Git
* GitHub

## Структура базы данных

В проекте используются следующие таблицы:

1. Rooms — номера
2. Guests — гости
3. Bookings — бронирования
4. Categories — категории номеров
5. Users — пользователи системы
6. Employees — сотрудники
7. Services — дополнительные услуги
8. Payments — платежи

Общее количество таблиц: 8.

## Реализованные операции CRUD

Для основных сущностей реализованы операции:

* Rooms
* Guests
* Users
* Employees

Поддерживаются операции создания, просмотра, изменения и удаления данных.

## Роли пользователей

### Администратор

* управление пользователями;
* управление сотрудниками;
* просмотр базы гостей;
* управление номерным фондом.

### Сотрудник ресепшена

* регистрация гостей;
* заселение и выселение;
* работа с номерами.

## Запуск проекта

1. Запустить проект HotelApi.
2. Дождаться запуска сервера по адресу:

http://localhost:5157

3. Запустить проект HotelDesktop.
4. Выполнить вход в систему.
